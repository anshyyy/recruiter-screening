# API (`@repo/api`)

NestJS HTTP service for the recruiter-screening monorepo.

- **Setup, env, and scripts:** see the [root README](../../README.md).
- **Modules, data model, integrations:** see [docs/BACKEND_ARCHITECTURE.md](../../docs/BACKEND_ARCHITECTURE.md).

**Quick reference**

| Item | Value |
|------|--------|
| Dev | `pnpm dev` (from `apps/api` or run `pnpm dev` from repo root) |
| Swagger | `http://localhost:8080/api/docs` |
| Global prefix | `/api` |
